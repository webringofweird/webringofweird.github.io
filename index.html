Join us!

Visit the repository website in GitHub, fork it, add your homepage details to the sites.json file, and send a PR. If your site is weird enough, we'll accept it!

Get the code!

Simply add this piece of code to your homepage to display the webring:

<script src="https://webringofweird.github.io/data/widget.js" data-url="{YOUR_URL}" async></script>

Make sure to specify your homepage's url with the data-url attribute!
